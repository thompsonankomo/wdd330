Accessing form elements

The legacy DOM has a useful attribute called document.forms that returns a collection of all forms in the document in the order that they appear in the mark up To return a collection of forms we have to use the index notation const form = document.forms[0] Sometimes instead of using a numerical index we can use the name attribute of the form in order to be able to identify it const form = document.forms.search; Normally in the above example square bracket notaion can be used to avoid confusion in declaring the form. const form = document.forms['search'];

This is probably the msot importnt event occuring when the form is submitted normally this will send the content of the form to the server to be processed.
